## 使い方

### GitHub Pages で公開する手順

1. このリポジトリを GitHub に作成（Public）
2. `index.html` をアップロード
3. リポジトリの **Settings → Pages** で
   - Source: `Deploy from a branch`
   - Branch: `main` / `(root)` を選択して Save
4. 数分後、`https://<ユーザー名>.github.io/<リポジトリ名>/` で公開される
5. URL を LINE で共有してみんなで見る 🎉

### ローカルで確認する場合

`index.html` をダブルクリックするだけでブラウザで開けます。
